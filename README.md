Smart To-Do List

A simple desktop task manager application built with Java Swing and AWT for our class project.

What it does

This app helps you manage your daily tasks. You can add tasks, set their priority and status, and track their deadlines.

Features

- Add, update, and delete tasks
- Set priority levels — Low, Medium, and High
- Set task status — Pending, In Progress, and Done
- Deadline tracking with color warnings
  - Red means the task is overdue
  - Orange means the deadline is within 3 days
- Save and load tasks so they don't disappear when you close the app
- Shows total, done, and pending task counts

Tools Used

- Java
- Java Swing (for the UI components)
- Java AWT (for layouts, colors, and fonts)

How to Run

1. Make sure you have Java installed on your computer
2. Open the project in VSCode
3. Install the Extension Pack for Java
4. Open `Main.java` and click the Run button

Files

- `Main.java` — starts the application
- `MainFrame.java` — the main window and UI
- `Task.java` — represents a single task
- `TaskManager.java` — handles adding, updating, and deleting tasks
